Semantic HTML

Semantic HTML Tags

Tag Name Description
<header> Header This tag represents the header or top portion of a web page
<nav> Navigation This tag contains a navigation bar that links to other parts of the web page
<section> Section This tag represents a major section of the web page to split up ideas
<article> Article In a website such as a blog this contains the main material for the page
<aside> Aside This contains additional information to the side of the main article
<figcaption> Figure Caption On any kind of figure this would be sub caption that gives a description of the figure
<figure> Figure This would be any kind of figure within a page such as graph or image
<footer> Footer The last section of a web page that contains further information or links
